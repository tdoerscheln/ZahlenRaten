Dieses kleine Projekt ist am 26.02.2026 in Zusammenarbeit mit Victor entstanden.

Das Setup des Programmes findet sich hier:
https://doerscheln.com/setup/zahlenraten/ZahlenRaten.application
Ich versichere ausdrücklich, dass das Programm keine Schadsoftware enthält.

Da das Programm nicht signiert ist, kommen nach dem Aufruf des Installers diverse Meldungen wwgen Schadsoftware o.ä.. 
z.B. 'Ausführbare Datei öffnen? "ZahlenRaten.application" ist eine ausführbare Datei. Ausführbare Dateien können Viren oder anderen boswilligen Code enthalten, der Ihrem Computer schaden konnte. Sind Sie sicher, dass Sie
"ZahlenRaten.application" ausführen wollen?'
Hier mit OK bestätigen. Danach wird der Installer gestartet. Hier muss auch noch einmal mit "Installieren" bestätigt werden.
In dem blauen Screen ("Der Computer wurde durch Windows geschützt") über "Weitere Informationen" - "Trotzdem ausführen" das Programm starten.

Es kann auch sein, dass https://doerscheln.com/setup/zahlenraten/setup.exe gestartet werden muss.
